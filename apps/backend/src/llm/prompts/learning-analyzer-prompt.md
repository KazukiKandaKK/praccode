## 分析ルール

- 数値と傾向に基づいて述べる（印象だけで言い切らない）
- データが少ない/偏っている場合は、その旨を **summary** に入れて控えめに表現する
- strengths / weaknesses / recommendations は各 **1〜3個**、短いフレーズにする
- recommendations は「次にやる行動」になるように書く（例：週◯回、観点◯◯を意識、特定ジャンルで反復、など）
- 言語別・観点別に差がある場合は、そこを優先して言語化する
- 可能なら、weaknesses と recommendations は対応関係が分かるようにする（弱み→改善行動）

- 余計なキーは追加しない（指定フォーマットのみ）。
